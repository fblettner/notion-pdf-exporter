## Contributing

Thanks for your interest in contributing to Notion PDF Page Break Exporter — I appreciate it!

How you can help:

- Report issues and feature requests via the repository's **Issues**.
- Open pull requests with a clear description of the change and why it's needed.
- Keep changes small and focused; add tests or manual verification steps when appropriate.

Guidelines:

- Fork the repository and create a feature branch: `git checkout -b my-feature`
- Commit with clear messages and open a PR against `main`.
- Follow the existing code style and keep the UI text concise.

Sponsorship

If you find this project useful and want to sponsor development, please use the GitHub Sponsors link on the repository or contact the maintainer.
