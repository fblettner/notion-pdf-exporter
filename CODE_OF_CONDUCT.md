# Contributor Code of Conduct

This project follows a standard Contributor Code of Conduct to foster an open and welcoming community.

Expected behavior:

- Be respectful and considerate in all interactions.
- Take care to avoid demeaning or exclusionary language.
- Be open to constructive feedback and discussion.

Unacceptable behavior includes, but is not limited to:

- Harassment, hate speech, or personal attacks.
- Disruptive behavior in issue threads or pull requests.

Enforcement and contact

If you experience or witness unacceptable behavior, please open an issue or contact the project maintainer (Franck Blettner) via the repository contact details so it can be addressed.

This project adapts the principles of the Contributor Covenant. For a full code of conduct, see: https://www.contributor-covenant.org/
